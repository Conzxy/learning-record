## 更改语言为英文等其他语言（即非中文）
打开Visual Studio Installer：

- 工具->获取工具和功能
- windows搜索"Visual Studio Installer"

点击语言包下载其他语言。
然后选项搜索“区域设置”，更改为其他语言。
